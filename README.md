# Amy's portfolio

A fast, responsive personal portfolio for a full-stack developer, AI engineer, and data engineer. It is built with plain HTML, CSS, and JavaScript and is ready to host for free on GitHub Pages.

## Preview locally

Open `index.html` directly, or run a small local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Personalise it

All portfolio copy is in `index.html`. Before sharing the site widely, update:

- the introductory biography and preferred professional title;
- the three cards in “Selected work” with real projects, outcomes, screenshots, and links;
- the GitHub-only contact links with your preferred email and LinkedIn profile;
- the skill tags so they name the technologies you use most;
- the page title, description, and social sharing metadata in `<head>`.

## Publish with GitHub Pages

For a repository named `amyyua.github.io`, push these files to the repository's default branch. In **Settings → Pages**, choose **Deploy from a branch**, select the default branch and `/ (root)`, then save. The site will be available at [https://amyyua.github.io](https://amyyua.github.io).
